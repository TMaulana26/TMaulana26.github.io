KILAS SINGKAT

Mengapa ada Web ini ?

Secara singkat website ini di buat untuk memenuhi UTS semester 6 saya. Dan juga dengan tugas ini saya bisa berlatih tentang HTML & Bootstrap 5.0.

Web ini dibuat pakai apa ?

Pebuatannya menggunakan HTML & PHP dengan di bantu oleh Bootstrap 5.0. sisanya saya mengode kan line baru atau pun saya mengedit line code yang sudah ada, agar sesuai.

Dan mengapa harus Ungu ?

Saat pembuatan konsep awalnya saya searching di mbah Google tentang color palette atau warna yang memiliki kombinasi yang bagus, & yang keluar adalah warna ungu ini.

IniWebsite di buat untuk Mengapresiasi warna
